# Dotfiles — Songster's Arch + Hyprland Setup

## Fresh Install Instructions

1. Clone this repo:
   git clone https://github.com/yourusername/dotfiles ~/dotfiles

2. Run the install script:
   cd ~/dotfiles
   ./install.sh

3. Reboot.

## What gets installed
- All pacman packages from packages.txt
- AUR packages: auto-cpufreq, brave-bin
- paru (AUR helper)
- All configs symlinked via GNU Stow

## Configs included
- Hyprland, Hyprlock, Hypridle, Hyprpaper
- Kitty terminal
- Waybar
- Neovim
- Rofi
- Btop
- Lazygit
- SwayNC notifications
- GTK 3.0 theme
- .zshrc
